# نظام إدارة المدرسة

نظام إدارة مدرسة باللغة العربية مع دعم RTL كامل

## الميزات:
- تسجيل الدخول للأدوار المختلفة (مدير، معلم، طالب، ولي الأمر)
- إدارة الطلاب (إضافة، تعديل، حذف، بطاقات QR)
- نظام الحضور والغياب
- إدارة الدرجات والتقارير
- رفع صور الطلاب
- طباعة التقارير

## المتطلبات:
- Python 3.7+
- Flask
- SQLAlchemy
- QRCode
